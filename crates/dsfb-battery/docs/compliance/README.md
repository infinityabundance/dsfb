# Compliance Support Layer

This directory contains crate-local standards mappings for `dsfb-battery`. These files are mapping and traceability artifacts only. They do not claim certification, approval, qualification, or regulatory acceptance.

The production mono-cell workflow remains unchanged. The optional compliance helper writes only under `outputs/compliance/...`.

Available mappings:

- `face_uop_mapping.md`
- `do178c_dal_mapping.md`
- `stc_support.md`
- `mil_std_882e_mapping.md`
- `rust_safety_subset.md`
- `nist_cmmc_mapping.md`
- `iec_61508_mapping.md`
- `iec_61131_3_mapping.md`
- `ieee_1547_mapping.md`
- `iso_26262_mapping.md`
- `industry_standards_mapping.md`
- `ieee_754_determinism.md`
- `iso_15926_mapping.md`
- `iso_25010_mapping.md`
- `sosa_mapping.md`
- `sotif_mapping.md`

The consolidated standards-status CSV is at `compliance/standards_status_matrix.csv`.
