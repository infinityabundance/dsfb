## Mathematical framework

| Construct | Symbol | Rust type | Source |
|-----------|--------|-----------|--------|
| Residual sign triple | σ(k) = (‖r(k)‖, ṙ(k), r̈(k)) | `ResidualSign` | §2.1 |
| Admissibility envelope | E = B(0, ρ_max) ⊆ V; δ-band ∂E | `AdmissibilityEnvelope` | Def 4.1 |
| Grammar FSM | δ: G × Σ → G (total) | `GrammarFsm` / `GrammarState` | Thm 3.1 |
| Endoductive operator | ℰ: Traj → Ep | `Enduce` trait | Cor 5.4 |
| Provenance tag | φ = (σ, g, α, range) | `ProvenanceTag` | Thm 8.3 |
| Episode | (m, φ): Motif × ProvenanceTag | `Episode` | Def 6.1 |
| Observer | 𝒪: Traj → Ep (pure) | `Observer<E>` | Thm 3.2 |
| Grammar fusion | G₁ ⋈ G₂ | `GrammarFusion` | Prop 7.1 |
| Heuristics bank | h: Motif ↦ [MotifPattern] | `HeuristicsBank` | §9 |

**Envelope constraint (Definition 4.1):** δ ≤ ρ_min / 4. Checked by the constructor;
panics statically rather than silently violating the geometric invariant.

**Totality (Theorem 3.1):** The grammar transition function δ is defined for *every*
(g, σ) pair. There are no undefined/exceptional paths through the FSM.

**Deterministic auditability (Theorem 8.3):** Any third party re-running ℰ with
(φ, h) on the same trajectory window reproduces the *identical* Episode. The
ProvenanceTag is a sufficient certificate.

## Safety-case properties (SC-1 – SC-6)

| Property | Claim | Rust enforcement mechanism |
|----------|-------|---------------------------|
| **SC-1** Determinism | `enduce` is a pure function of its inputs | Return type `Episode` (not `Option`, not `Result`) |
| **SC-2** Non-Interference | Observer does not mutate observed system | `Observer` holds no `&mut` to host state; Rust borrow checker |
| **SC-3** Auditability | Every Episode carries a recoverable certificate | `Episode.provenance: ProvenanceTag` always present |
| **SC-4** Coverage | Grammar classifies every timestep | `GrammarFsm::step` is a total function |
| **SC-5** No Silent Failure | Unrecognized events are labelled, not dropped | `Motif::Unknown` with complete ProvenanceTag — never `None` |
| **SC-6** Graceful Degradation | Impulsive inputs do not panic | `Unknown` episodes produced; no `unwrap()` in hot path |

## Quick start

```rust
use dsfb_semiotics_calculus::{
    AdmissibilityEnvelope, HeuristicsBank, Observer
};

// ρ_min=0.1, ρ_max=1.0, δ=0.02 — constraint: δ ≤ ρ_min/4 checked at construction
let envelope = AdmissibilityEnvelope::new(0.1, 1.0, 0.02);

// Day-One deployment — empty bank is valid (Proposition 9.1)
let bank = HeuristicsBank::new();

let mut observer = Observer::new(envelope, bank);

let residuals = vec![0.1, 0.3, 0.5, 0.7, 0.85, 0.96, 1.1];
let episodes = observer.observe_trajectory(&residuals);

for ep in &episodes {
    println!("{}", ep);
}
// → Episode(Unknown, steps 0–6, provenance: …)
```

**Day-One proposition (Prop. 9.1):** An empty bank is operationally valid.
The system discovers and characterises structural events before any fault library is
built. Every excursion produces `Motif::Unknown` *with complete ProvenanceTag* —
usable audit data from the first deployment second.

## Library API

### `AdmissibilityEnvelope`

```rust
pub struct AdmissibilityEnvelope {
    pub rho_min: f64,  // inner exclusion radius
    pub rho_max: f64,  // outer boundary (admissible iff ‖r‖ ≤ ρ_max)
    pub delta: f64,    // boundary half-width; δ ≤ ρ_min/4 enforced
}
impl AdmissibilityEnvelope {
    pub fn new(rho_min: f64, rho_max: f64, delta: f64) -> Self
    pub fn classify(&self, magnitude: f64) -> EnvelopeRegion
}
```

### `GrammarFsm` / `GrammarState`

```rust
pub enum GrammarState { Admissible, Boundary, Violation }

pub struct GrammarFsm { pub state: GrammarState }
impl GrammarFsm {
    pub fn new() -> Self
    pub fn step(&mut self, region: EnvelopeRegion) -> GrammarState
    // total — no undefined path
}
```

### `Observer<E: Enduce>`

```rust
pub struct Observer<E: Enduce> {
    pub envelope: AdmissibilityEnvelope,
    pub bank: HeuristicsBank,
    // no &mut to host system
}
impl<E: Enduce> Observer<E> {
    pub fn observe_trajectory(&mut self, residuals: &[f64]) -> Vec<Episode>
}
```

### `Enduce` trait

```rust
pub trait Enduce {
    fn enduce(
        &self,
        signs: &[ResidualSign],
        grammar_path: &[GrammarState],
        bank: &HeuristicsBank,
        step_range: (usize, usize),
    ) -> Episode;   // total — never panics, never returns Option
}
```

### `HeuristicsBank`

```rust
pub struct HeuristicsBank { /* monotone map */ }
impl HeuristicsBank {
    pub fn new() -> Self
    pub fn augment(&mut self, motif: Motif, pattern: MotifPattern)
    // no remove() — bank is monotone by construction (Theorem 7.2)
}
```

### `Episode` / `ProvenanceTag`

```rust
pub struct Episode {
    pub motif: Motif,
    pub provenance: ProvenanceTag,
}

pub struct ProvenanceTag {
    pub sign_sequence: Vec<ResidualSign>,
    pub grammar_path: Vec<GrammarState>,
    pub add_descriptor: String,   // algebraic descriptor α
    pub step_range: (usize, usize),
}
```

### `GrammarFusion`

```rust
pub struct GrammarFusion { pub g1: GrammarFsm, pub g2: GrammarFsm }
impl GrammarFusion {
    pub fn step_both(&mut self, r1: f64, r2: f64) -> (GrammarState, GrammarState)
}
```
